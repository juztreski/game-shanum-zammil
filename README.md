# Shanum & Zammil - Game Anak 2 Player 🎈🦖

Game web interaktif split-screen gratis yang dirancang khusus untuk balita (anak di bawah 3 tahun) dimainkan berdua menggunakan satu layar (ponsel atau tablet).

## 🚀 Fitur Utama

- **5 Pilihan Permainan Interaktif**:
  1. 🎈 **Tepok Balon (Balloon Pop)**: Memecahkan balon melayang untuk mencetak skor.
  2. 🍎 **Beri Makan (Feeding Game)**: Menyuapi makanan lezat langsung ke mulut karakter.
  3. ⭐ **Tangkap Bintang (Catch the Star)**: Ketuk bintang ceria yang bermunculan secara acak.
  4. 🥚 **Ketuk Telur (Tap the Dino Egg)**: Ketok telur 5 kali sampai menetas menjadi bayi hewan lucu (+5 poin!).
  5. 🎵 **Suara Hewan (Soundboard)**: Memainkan musik suara kucing, anjing, bebek, dan katak yang membuat karakter menari.
- **Sistem Kemenangan 30 Poin**: Pemain pertama yang mencapai 30 poin menang dan mendapatkan perayaan confetti bintang serta musik victory!
- **Split-Screen Ganda**: Layar terbagi menjadi dua area kontrol independen (Panel Atas untuk Shanum 💖, Panel Bawah untuk Zammil 🦖).
- **Putar Layar 180°**: Memutar panel atas agar dua anak bisa duduk berhadapan saat bermain bersama di satu meja.
- **Efek Suara Instan**: Dibuat langsung dengan Web Audio API (suara balon meletus, kunyahan makanan, instrumen musik hewan) tanpa loading internet.

## Cara Bermain Secara Lokal

Jika Anda ingin menjalankannya secara lokal di komputer Anda:
1. Pastikan Node.js terinstal.
2. Jalankan perintah berikut di terminal:
   ```bash
   node server.js
   ```
3. Buka browser di alamat: `http://localhost:8080`
