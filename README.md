# Shanum & Zammil - Game Anak 2 Player 🎈🦖

Game web interaktif split-screen gratis yang dirancang khusus untuk balita (anak di bawah 3 tahun) dimainkan berdua menggunakan satu layar (ponsel atau tablet).

## Fitur Utama

- **Split-Screen Ganda**: Layar terbagi menjadi dua area kontrol independen (Atas untuk Shanum, Bawah untuk Zammil).
- **Putar Layar 180°**: Fitur unik untuk memutar panel atas agar dua anak bisa duduk berhadapan saat bermain bersama di satu meja.
- **Efek Suara Instan**: Dibuat langsung dengan Web Audio API (suara balon meletus, kunyahan makanan, instrumen musik hewan) tanpa loading internet.
- **Tiga Pilihan Permainan**:
  1. 🎈 **Tepok Balon**: Memecahkan balon-balon yang melayang.
  2. 🍎 **Beri Makan**: Menyuapi makanan lezat ke karakter Shanum & Zammil.
  3. 🎵 **Suara Hewan**: Soundboard interaktif yang memicu tarian karakter dan suara lucu.

## Cara Bermain di HP / Tablet (Lokal)

Jika Anda ingin menjalankannya secara lokal:
1. Pastikan Node.js terinstal.
2. Jalankan perintah berikut di terminal:
   ```bash
   node server.js
   ```
3. Buka browser di alamat: `http://localhost:8080`
